# AutoClickMouse
# 自动点击鼠标
1、首先，确保你已经安装了 pynput 库
>pip install pynput

2、运行脚本 mouse.Listener.py 获取需要鼠标点击位置的坐标；

3、打开 auto_click_mouse.py 进行编辑
> 设定需要鼠标点击位置的坐标点 (x, y)

> 设定点击间隔时间（秒）示例为1s:
> 
>click_interval = 1

> 设定点击次数（设置为 -1 表示无限次）示例为10次:
> 
>click_count = 10  

4、保存并运行脚本 auto_click_mouse.py ;

5、需要停止运行可使用 Ctrl+C 或 关闭终端窗口。
